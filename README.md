proof of dual booting and some basic Linux command
1. ls
   It lists all the files and folders in current spot.

2. cd — "Take me there"
Short form of Change Directory.used to change from one directory to other

3. mkdir 
 stands for Make Directory.

4. rm 
This is the Remove command. Use it to delete files.

5. pwd 
It stands for Print Working Directory.

  6.rm

7. cp

8. mv

**MY LEARNING NOTES**
Partitioning: Carving out space in the SSD without nuking my Windows data. (Terrifying the first time!)

The Live USB: Using tools like Rufus or BalenaEtcher to create a "portal" into Linux.

BIOS/UEFI Tweaks: Disabling Secure Boot and Fast Boot—the "gatekeepers" that try to keep you on Windows.

Challenges Faced 

The Bootloader Battle: Occasionally, Windows Updates decide they own the house and "hide" the Linux Grub menu. Learning to repair the bootloader was my first "real" Linux lesson.

Driver Drama: My Wi-Fi card didn't work out of the box. I had to learn how to tether my phone's internet to download the proprietary drivers. It felt like MacGyver-ing my laptop.

The "Case-Sensitive" Trap: Coming from Windows, I didn't realize Folder and folder were different things. I spent 20 minutes looking for a file that was right in front of me.

Permissions (sudo): Getting "Permission Denied" errors until I realized that in Linux, you have to explicitly ask for "God Mode" (sudo) to do anything important.
My Perspective
In Windows, if something breaks, you wait for a patch. In Linux, if something breaks, you are the patch. There is a specific kind of "hacker high" you get when you fix a broken screen resolution or a sound driver by typing three lines of code into a black box. It makes the computer feel like yours, not a service you're renting from a corporation.

**Conclusion**
Linux is a steep climb, but the view from the top is worth it. It teaches you patience, logic, and how computers actually "think."

My advice? Don't be afraid to break things. That’s why we dual-boot—Windows is there for safety, but Linux is where the growth happens. If you're looking for total control over your digital life, this is the only way to go.
